# opacity-picker


## usage
`op_picker.init('element-id');`


## methods
```
  init(id)
  // initiate op_picker (idContainer required)
  
  getColor()
  // get current bar color
  
  setColor(color)
  // set bar color
  
  getOpacity()
  // get current opacity value 
  
  setOpacity(value)
  // set numerical opacity value [0, 1]
  
  onChange(opacity)
  // fired on opacity change
  ```
